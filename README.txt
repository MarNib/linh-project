#This module resizes and enhances contrast (equalizing histogram) of a batch of images, located in the same folder.

#To use the module image_proce.py:

- Open terminal
- Go to the same directory where the python module is located
- Call the following command: python image_proce.py -d 'PATH_TO_IMAGES' -w 256 -h 256 
	(arguments can be modified as the user requires)  

#The code used in this module was adapted from: http://pythoncentral.io/resize-image-python-batch/
